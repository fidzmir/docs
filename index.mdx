TCBot is a WhatsApp-powered automation assistant built for manufacturing production teams. Operators on the floor can create and close TPM red tags, scan product labels using AI-powered OCR, query live production data, and receive automated shift reports — all from within WhatsApp, with no separate app required.

## How It Works

<Steps>
  <Step title="Operator opens WhatsApp">
    Your team uses the same WhatsApp they already have. No additional apps, no logins.
  </Step>
  <Step title="Send a bot command">
    Type a command like `/am`, `/input 33G198`, or `/openlist` to start a workflow.
  </Step>
  <Step title="Bot guides the interaction">
    TCBot walks the operator through each step — collecting data, photos, and confirmations.
  </Step>
  <Step title="Data saved to Google Sheets">
    Completed entries are written directly to your connected Google Sheets in real time.
  </Step>
</Steps>

<Note>
  TCBot is administered by your operations team. If you're an operator looking to get started, see the [Quick Start](/quickstart) guide.
</Note>
