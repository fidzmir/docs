---

The `/am` command starts a guided red tag creation flow directly in WhatsApp. TCBot collects all required information step by step and saves the completed record to your Google Sheets automatically — no manual entry needed.

## Workflow

<Steps>
  <Step title="Start the flow">
    Type `/am` in the WhatsApp chat. TCBot greets you by name and presents the sheet selection menu.

    ```
    You: /am
    TCBot: Halo [Name]!
           Pilih Sheet:
           1. Produksi HPL
           2. Produksi Adhesive
           3. Produksi Flooring
           4. Produksi PVC Cikupa
    ```
  </Step>

  <Step title="Select your sheet">
    Reply with the number that matches your production line.

    | Number | Sheet |
    |--------|-------|
    | `1` | Produksi HPL |
    | `2` | Produksi Adhesive |
    | `3` | Produksi Flooring |
    | `4` | Produksi PVC Cikupa |

    ```
    You: 1
    TCBot: ✅ Sheet: Produksi HPL
           2. Kode Dept (HPL/ADH/FLR/PVC):
    ```
  </Step>

  <Step title="Enter your department code">
    Type your department code: `HPL`, `ADH`, `FLR`, or `PVC`.

    ```
    You: HPL
    TCBot: ✅ Dept: HPL
           3. Nama Mesin:
    ```
  </Step>

  <Step title="Enter the machine name">
    Type the name of the machine where the issue was found (e.g., `Calender`, `Mixing Tank`).

    ```
    You: Calender
    TCBot: ✅ Mesin: Calender
           4. Pilih Abnormality (Contoh: 1,3):
           1. Bocor
           2. Usang
           3. Rusak
           4. Kendur
           5. Hilang
           6. Cacat
           7. Lain-Lain
           0. Kosong
    ```
  </Step>

  <Step title="Select abnormality types">
    Type one or more numbers to identify the abnormality. Separate multiple selections with a comma or a space.

    | Number | Abnormality |
    |--------|-------------|
    | `1` | Bocor |
    | `2` | Usang |
    | `3` | Rusak |
    | `4` | Kendur |
    | `5` | Hilang |
    | `6` | Cacat |
    | `7` | Lain-Lain |
    | `0` | Kosong (none) |

    <Tip>
      Both `1,3` and `1 3` are valid — use whichever format feels faster. The same multi-select format applies to contamination and hard-to-access options.
    </Tip>
  </Step>

  <Step title="Select contamination types">
    Type one or more numbers to identify any contamination present. Use `0` if there is none.

    | Number | Contamination |
    |--------|---------------|
    | `1` | Pelumas |
    | `2` | Air/Cairan |
    | `3` | Produk |
    | `4` | Limbah |
    | `5` | Kotoran |
    | `6` | Korosi |
    | `0` | Kosong (none) |
  </Step>

  <Step title="Select hard-to-access issues">
    Type one or more numbers to identify any hard-to-access maintenance tasks. Use `0` if there are none.

    | Number | Hard to Access |
    |--------|----------------|
    | `1` | Membersihkan |
    | `2` | Memeriksa |
    | `3` | Melumasi |
    | `4` | Mengganti |
    | `5` | Mengencangkan |
    | `0` | Kosong (none) |
  </Step>

  <Step title="Enter a description">
    Type a short description of the finding in your own words (e.g., `Oil leak on left bearing`).
  </Step>

  <Step title="Send a photo">
    Send a photo of the issue directly in the WhatsApp chat.

    <Warning>
      Only image photos are accepted at this step. Sending a document file, video, or plain text will cause TCBot to ask you to retry with a valid photo.
    </Warning>
  </Step>

  <Step title="Review and confirm">
    TCBot displays a full summary of your entries. Review each field carefully.

    - Type `1` to **submit** the tag.
    - Type `2` to **cancel** without saving.

    On a successful submission, TCBot confirms the record has been saved to your Google Sheet.

    ```
    TCBot: ✅ BERHASIL! Data Red Tag telah tercatat.
    ```

    You can view the tag immediately with `/open [TAG_CODE]` or find it in `/openlist`.
  </Step>
</Steps>
