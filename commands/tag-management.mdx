TCBot gives you instant access to your TPM tag registry from WhatsApp. List all open tags, inspect individual tag details, and close resolved issues — all without leaving the chat.

---

## List All Open Tags — `/openlist`

**Syntax:** `/openlist`

Send `/openlist` to retrieve every tag currently in **OPEN** status. TCBot returns a numbered list with each tag's code and a brief description.

```text
You: /openlist
TCBot: 📋 DAFTAR TAG OPEN:

       1. AM-HPL-0001 - Bocor, Mesin Calender
       2. AM-ADH-0003 - Rusak, Mixing Tank
```

Use the tag codes from this list with `/open` or `/close`.

---

## View Tag Details — `/open [TAG_CODE]`

**Syntax:** `/open AM-HPL-0001`

Send `/open` followed by a tag code to retrieve its full record. TCBot returns all fields entered at creation time — and if a photo was attached when the tag was created, TCBot sends it as well.

```text
You: /open AM-HPL-0001
TCBot: 📄 DETAIL TAG: AM-HPL-0001

       Status: OPEN
       Tanggal: 01/06/2025
       Mesin: Calender
       Pelapor: Ahmad
       Abnormality: Bocor
       Contamination: Pelumas
       Hard to Access: Membersihkan
       Deskripsi: Oil leak on left bearing
```

<Note>
  If the tag code you enter is not found in the database, TCBot will notify you. Double-check the code by running `/openlist` first.
</Note>

---

## Close a Tag — `/close [TAG_CODE]`

**Syntax:** `/close AM-HPL-0001`

Send `/close` followed by a tag code to begin the closure flow. TCBot requires photo evidence of the completed repair before updating the tag's status.

<Steps>
  <Step title="Start the closure">
    Type `/close` followed by the tag code.

    ```text
    You: /close AM-HPL-0001
    TCBot: ✅ Process Tutup Tag: AM-HPL-0001

           Silakan kirimkan FOTO BUKTI perbaikan untuk menutup tag ini.
           (Atau ketik /cancel untuk membatalkan)
    ```
  </Step>

  <Step title="Send a repair photo">
    Take and send a photo showing the resolved issue directly in the WhatsApp chat.

    <Warning>
      Only image photos are accepted as proof. Sending a document file, video, or plain text will cause TCBot to ask you to retry with a valid photo.
    </Warning>
  </Step>

  <Step title="Review and confirm">
    TCBot shows a closure confirmation summary.

    ```text
    TCBot: 📝 KONFIRMASI TUTUP TAG

           Tag: AM-HPL-0001

           1. Konfirmasi Tutup
           2. Batal
    ```

    - Type `1` to **confirm** and close the tag.
    - Type `2` to **cancel** and leave the tag open.
  </Step>

  <Step title="Closure confirmed">
    On success, TCBot updates the tag's status to **CLOSE** in Google Sheets and confirms the change.

    ```text
    TCBot: ✅ BERHASIL! Tag AM-HPL-0001 telah ditutup (CLOSE).
    ```

    If the tag code was not found in the database, TCBot returns an error instead.

    ```text
    TCBot: ❌ GAGAL! Tag AM-HPL-0001 tidak ditemukan di database.
    ```
  </Step>
</Steps>

<Note>
  If the tag code is not found, verify the correct code with `/openlist` and try again.
</Note>
