TCBot responds to typed commands in WhatsApp. All commands start with `/`. Send any command at any time — and use `/cancel` whenever you need to exit a flow and start fresh.

## Quick Reference

| Command | Description |
|---|---|
| `/am` | Create a new TPM Red Tag |
| `/openlist` | List all open tags |
| `/open [TAG_CODE]` | View tag details |
| `/close [TAG_CODE]` | Close a tag with photo |
| `/input [ITEM_CODE]` | Scan a product label with AI |
| `/cek [query]` | Query production spreadsheet |
| `/ngobrol [DEPT] [question]` | Chat with AI about production data |
| `/cancel` | Cancel any active workflow |

<Note>
  Commands are case-insensitive. `/AM`, `/am`, and `/Am` all work the same way.
</Note>

<Tip>
  Use `/cancel` at any step in a multi-step flow — such as `/am` or `/close` — to safely exit without saving any data. You can then start over with a fresh command.
</Tip>
