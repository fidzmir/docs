---
title: "Get Started with TCBot in 5 Minutes"
sidebarTitle: "Quick Start"
description: "Learn how to send your first TCBot command on WhatsApp and complete a sample red tag workflow from start to finish."
---

You only need WhatsApp to use TCBot — no app downloads, no account registration. This guide walks you through sending your first commands and completing a full red tag creation workflow from start to finish.

<Steps>
  <Step title="Get the bot's WhatsApp number from your administrator">
    TCBot runs on a WhatsApp number that your administrator sets up and manages. Ask your admin for the bot's number before you begin. You cannot discover it automatically — it must be provided to you.
  </Step>

  <Step title="Save the number in your contacts">
    WhatsApp requires a number to be saved as a contact before you can send messages to it. Add the bot's number to your phone contacts with a recognizable name such as **TCBot** or **Production Bot**. Then open WhatsApp and confirm the contact appears in your chat list.
  </Step>

  <Step title="Send /openlist to verify your connection">
    Open the TCBot chat and send your first command:

    ```
    /openlist
    ```

    TCBot replies with a list of all currently open TPM tags. This is a safe, read-only command — it changes nothing and is the fastest way to confirm the bot is responding to your messages.
  </Step>

  <Step title="Send /am to create your first red tag">
    Send the `/am` command to start a guided red tag creation flow. TCBot asks you a series of short questions — answer each one by replying with the number or text it requests.

    Here is a complete sample conversation:

    ```
    You:    /am

    TCBot:  Halo [Name]!
            Pilih Sheet:
            1. Produksi HPL
            2. Produksi Adhesive
            3. Produksi Flooring
            4. Produksi PVC Cikupa

    You:    1

    TCBot:  ✅ Sheet: Produksi HPL

            2. Kode Dept (HPL/ADH/FLR/PVC):

    You:    HPL

    TCBot:  ✅ Dept: HPL

            3. Nama Mesin:

    You:    Press Line 3

    TCBot:  ✅ Mesin: Press Line 3

            4. Pilih Abnormality (Contoh: 1,3):
            1. Bocor
            2. Usang
            3. Rusak
            4. Kendur
            5. Hilang
            6. Cacat
            7. Lain-Lain
            0. Kosong

    You:    3

    TCBot:  ✅ Abnormal: Rusak

            5. Pilih Contamination:
            1. Pelumas
            2. Air/Cairan
            3. Produk
            4. Limbah
            5. Kotoran
            6. Korosi
            0. Kosong

    You:    0

    TCBot:  ✅ Contam:

            6. Pilih Hard To Access:
            1. Membersihkan
            2. Memeriksa
            3. Melumasi
            4. Mengganti
            5. Mengencangkan
            0. Kosong

    You:    4

    TCBot:  ✅ Access: Mengganti

            7. Deskripsi Singkat:

    You:    Roller aus, perlu penggantian segera

    TCBot:  ✅ Desk: Roller aus, perlu penggantian segera

            8. Kirim Foto Temuan:

    You:    [sends photo]

    TCBot:  📝 KONFIRMASI RED TAG

            Sheet: Produksi HPL
            Dept: HPL
            Mesin: Press Line 3
            Abnormality: Rusak
            Contamination: -
            Hard To Access: Mengganti
            Deskripsi: Roller aus, perlu penggantian segera

            1. Kirim
            2. Batal

    You:    1

    TCBot:  ✅ BERHASIL! Data Red Tag telah tercatat.
    ```

    For steps 4–6 (Abnormality, Contamination, Hard To Access), you can select multiple options by separating numbers with commas — for example, reply `1,3` to select the first and third items. Reply `0` to leave a field empty.
  </Step>
</Steps>

<Tip>
  At any point during a multi-step workflow — such as `/am` or `/input` — send `/cancel` to exit immediately. TCBot discards any partial input and returns to standby. Use this whenever you start a workflow by mistake or need to start over.
</Tip>

<Note>
  When you confirm a red tag submission, TCBot saves the data directly to your team's connected Google Sheets. You do not need to do anything else — the record is immediately visible to supervisors and maintenance staff with access to the sheet.
</Note>
